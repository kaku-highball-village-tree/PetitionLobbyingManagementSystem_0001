# TEXT FILE SPECIFICATION

## 概要

本ドキュメントは、

1相談
=
1TXT

として生成されるTXTファイル仕様を定義します。

TXTファイルは、

- 秀丸grep
- AI/RAG
- 長期保存
- PDF連携
- 将来OCR
- 1件単位管理

などを目的として生成されます。

---

# 基本思想

## TXT文化重視

本システムでは：

1相談
=
1TXT

を重視します。

---

## 理由

TXTは：

- grepに強い
- AIに強い
- 壊れにくい
- PDFと組み合わせやすい
- 長期保存に強い
- SQLite移行しやすい

ため。

# 禁止文字仕様

以下文字は：

```text
\ / : * ? " < > |
```

使用禁止。

---

## 置換仕様

禁止文字は：

```text
_
```

へ置換。

---

## 例

```text
道路/騒音
```

↓

```text
道路_騒音
```

# 画像置換後TXT仕様（追加）

## 対象

画像（.jpg / .jpeg / .png）からChatGPTで文字起こしして生成するTXTも、
**本ドキュメントと同一のTXT内容形式**を採用する。

---

## 帳票レイアウト由来の抽出項目

相談メモ画像レイアウト上、以下を優先抽出する。

- タイトル: `相談 メモ`
- ヘッダ1: `相談日` / `相談者` / `相談内容`
- ヘッダ2: `役所依頼日` / `依頼課` / `受付担当者` / `処理の可否`（日付 + 2項目）
- 本文欄: 長文可

判読不能な箇所は `[判読不能]` とする。
