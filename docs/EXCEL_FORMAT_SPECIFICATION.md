## 概要

本ドキュメントは、

```text

```

のExcel仕様を定義します。

本Excelファイルは：

- 市民相談
- 陳情
- 請願
- ロビー活動

などの管理入力用として使用します。


---

# 列定義

## A列

### 項目名

管理番号

### 内容

全体通し番号。

### 表示形式

ユーザー定義：

```text
000000
```

### 数式

A2：

手入力。

A3：

```excel
=A2+1
```

---

## B列

### 項目名

受付番号

### 内容

年月付き受付番号。

### B2

```excel
=TEXT(C2,"yyyy-mm")&"-0001"
```

### B3以降

```excel
=LEFT(B2,8)&TEXT(VALUE(RIGHT(B2,4))+1,"0000")
```

---

## G列

### 項目名

依頼課

### 入力方式

プルダウン。

### データ入力規則

入力値の種類：

```text
リスト
```

元の値：

```excel
=依頼課!$A$1:$A$200
```
